# AR Tsumiki

AR Tsumikiは、Three.jsを利用して開発されたWebベースのAR(拡張現実)アプリケーションです。ユーザーが物理空間でバーチャルなブロックとインタラクションできることが特徴です。

## デモ
[デモページ](https://code4fukui.github.io/ar-tsumiki/)で試用できます。

## 機能
- 携帯デバイスのカメラを使ったAR操作に対応
- 様々な形状と色のブロックを表示
- ブロックの移動、回転、スナップが可能
- ジグ(パズル)モードも用意

## 必要環境
WebXRに対応した最新のGoogle ChromeやMozilla Firefoxなど、スマートフォンやタブレットのブラウザが必要です。

## 使い方
デモページをモバイルデバイスのブラウザで開くと、カメラ使用の許可を求められます。その後、仮想のブロックを物理空間で操作できます。
