# AR Tsumiki

https://code4fukui.github.io/ar-tsumiki/ でデモを公開しています。

## デモ
3DオブジェクトをARで操作できるデモアプリです。オブジェクトの素材や色、表示設定を変更できます。

## 機能
- ARでオブジェクトをつかんで操作できる
- オブジェクトの素材や色、表示設定を変更可能
- 様々な形状のオブジェクトを表示できる

## 必要環境
WebARに対応したスマートフォンやタブレットが必要です。

## 使い方
スマートフォンやタブレットのブラウザでデモページを開くと、ARモードで体験できます。オブジェクトをつかんで移動したり、操作パネルから設定を変更できます。
